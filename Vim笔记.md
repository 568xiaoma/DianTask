vim笔记
===
***
**安装配置**
```
安装vim 命令:sudo apk-get install vim-gtk
配置vim 命令:sudo vim /etc/vim/vimrc(对vimrc文件编辑)
	1.打开高亮:检查syntax on是否被注释，如果注释，解注释.
	2.左侧行号:set nu
	3.tab长度设置为4:set tabstop
	4.覆盖文件时不备份:set nobackup
	5.突出显示当前行:set cursorline
	6.在右下角显示光标位置的状态行:set ruler
	7.自动缩进:set autoindent
```
***
**操作命令**
######保存与退出
```
Esc:退出当前状态，进入命令模式
w:写入文件
w file:另存为
w!:强制保存，不退出
wq:保存文件并退出
ZZ:保存文件并退出
wq!:强制保存文件，并退出
q:不保存文件，退出
q!:不保存文件强制退出
e!:放弃所有修改，从上次保存文件开始编辑
```
######光标跳转
```
v:与正常鼠标操作相似，光标经过地方会被选中
V:光标经过的行会被选中
ctrl v:选中光标起始与终点的矩形区域
gg:光标跳到第一行行首
G:光标跳到最后一行行首
numG:光标跳转到第num行
%:括号的寻找与跳转
num%:按百分比跳转到相应位置
$:移动到行尾
^:移动到行首(非空白字符)
0:移动到行首
/字符串:查找字符串
n:查找下一个
numn:查找接下来第三个
/^字符串:查找行首字符串
/字符串$:查找行尾字符串
/\.:查找特殊字符时用转移方法
```
######复制粘贴
```
y:复制到寄存器
yy:复制行到寄存器
p:从寄存器粘贴
"+y:复制到系统剪贴板
"+p:从系统剪贴板粘贴
```
######编辑操作
```
x:删除字符
dd:删除整行
D:删除光标后该行剩下内容
J:将下一行与本行合并空格隔开
u:撤销
U:重做(撤销撤销)
ctrl R:同上
i:进入insret模式
a:进入insert模式(光标会向后移动一位)
o:下面另起一行(进入insert模式)
O:上面另起一行(进入insert模式)
c:等同于delete与insert之和
cc:修改整行内容
```
######浏览操作
```
ctrl D:向下滚动半屏
ctrl U:向上滚动半屏
ctrl F:向下滚动一屏
ctrl B:向上滚动一屏
ctrl E:向下滚动一行
ctrl Y:向上滚动一行
```
